Sobre o projeto

Este projeto foi desenvolvido para transformar dados abertos do Ministério do Turismo em ferramentas úteis para agências de viagem. Nosso objetivo é facilitar a análise de mercado turístico e ajudar empresas a tomar decisões baseadas em dados reais.
